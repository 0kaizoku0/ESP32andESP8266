Visual Studio Code

Descargar Visual Studio Code del siguiente enlace https://code.visualstudio.com/download.

Selecciona la versión de 64 bits preferentemente. Si tienes permisos de administrador en tu computadora, selecciona System Installer, si no, hay que seleccionar User Installer.

 

Una vez descargado abre la aplicación con doble clic.

 

Se abrirá la siguiente ventana, donde aceptaremos los términos de licencia y daremos clic en siguiente varias veces hasta llegar a Instalar.

 

El programa comenzará a instalarse.

Una vez terminado podemos ejecutarlo dejando la opción marcada y dando clic en Finalizar.
